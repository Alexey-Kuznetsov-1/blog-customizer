# Проектная работа: Кастомизация страницы блога

## Описание проекта
Проект представляет собой страницу блога с возможностью кастомизации через выдвижную панель настроек. Пользователь может изменять шрифт, размер текста, цвета и ширину контента, а изменения применяются через CSS-переменные.

## Функциональные требования

- ✅ При нажатии на «стрелку» открывается сайдбар с настройками, при повторном нажатии или клике вне сайдбар закрывается.
- ✅ При изменении настроек в сайдбаре они не применяются сразу.
- ✅ После нажатия на «применить» стили применяются к статье.
- ✅ При нажатии «сбросить» настройки в форме сбрасываются на начальные, которые были при открытии страницы, и стили применяются к статье.
- ✅ Настройки устанавливаются через CSS-переменные, которые уже есть в стилях и установлены в коде в дефолтные значения.

Компоненты
ArticleParamsForm
Основной компонент формы настроек. Содержит:

Выпадающий список для выбора шрифта

Радио-группу для выбора размера шрифта

Выпадающие списки для цветов и ширины контента

Кнопки "Применить" и "Сбросить"

Article
Компонент статьи, который отображает контент и применяет выбранные стили через CSS-переменные.

CSS-переменные
В проекте используются следующие CSS-переменные:

css
--font-family     /* семейство шрифта */
--font-size       /* размер шрифта */
--font-color      /* цвет текста */
--bg-color        /* цвет фона */
--container-width /* ширина контейнера */

Критерии проверки
Сайдбар открывается по клику на стрелку

Сайдбар закрывается по повторному клику или клику вне области

Изменения в форме не применяются до нажатия "Применить"

Кнопка "Применить" применяет все выбранные настройки

Все настройки работают через CSS-переменные

Код написан на TypeScript без использования any
